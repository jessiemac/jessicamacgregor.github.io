A web platform designed and illustrated to reflect the iconic fluxus artist, Joseph Beuys. This website presents his manifesto using illustrative iconography and javascript. Each illustration was constructed using cut paper then transferred digitally into vector illustrations. Please visit the website to see the full use of html, CSS and Javascript.
